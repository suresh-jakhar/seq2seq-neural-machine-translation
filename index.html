French Translation API Documentation

Neural Machine Translation API for English to French

API Base URL

https://french-translation.onrender.com

Use this base URL for all API requests.

Authentication

Currently, this API does not require authentication. API key support will be added in future versions.

Supported Endpoint

POST/translate

Translates English text to French using a trained neural machine translation model.

Request Format

Headers

  • Content-Type: application/json (Required)

Request Body

Send a JSON object with the following structure:

{
  "text": "Your English text here"
}

Parameters

Parameter Type Required Description
text string Yes English text to translate (max 500 characters recommended)

Response Format

Success Response (200 OK)

{
  "success": true,
  "translated_text": "Votre texte fran√ßais ici",
  "original_text": "Your English text here"
}
Response Fields:
  • success: Boolean indicating translation status
  • translated_text: The French translation
  • original_text: The original English input

Error Responses

400 Bad Request - Invalid Input

{
  "success": false,
  "message": "No text provided"
}

Common causes:

  • Missing text field in request body
  • Empty string provided
  • Invalid JSON format
  • Missing Content-Type header

500 Internal Server Error

{
  "success": false,
  "message": "Translation failed"
}

Common causes:

  • Model processing error
  • Server overload
  • Text exceeds maximum supported length

Example Requests

Python (using requests)

import requests
import json

# API endpoint
url = "https://french-translation.onrender.com/translate"

# Request payload
payload = {
    "text": "Hello, how are you?"
}

# Headers
headers = {
    "Content-Type": "application/json"
}

# Make POST request
response = requests.post(url, json=payload, headers=headers)

# Parse response
if response.status_code == 200:
    data = response.json()
    print(f"Translation: {data['translated_text']}")
else:
    print(f"Error: {response.status_code}")

JavaScript (fetch API)

// API endpoint
const url = 'https://french-translation.onrender.com/translate';

// Request payload
const payload = {
  text: 'Hello, how are you?'
};

// Make POST request
fetch(url, {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify(payload)
})
  .then(response => response.json())
  .then(data => {
    if (data.success) {
      console.log('Translation:', data.translated_text);
    } else {
      console.error('Error:', data.message);
    }
  })
  .catch(error => console.error('Request failed:', error));

cURL

curl -X POST https://french-translation.onrender.com/translate \
  -H "Content-Type: application/json" \
  -d '{"text": "Hello, how are you?"}'

Node.js (axios)

const axios = require('axios');

async function translateText(text) {
  try {
    const response = await axios.post(
      'https://french-translation.onrender.com/translate',
      { text: text },
      { headers: { 'Content-Type': 'application/json' } }
    );
    
    return response.data.translated_text;
  } catch (error) {
    console.error('Translation error:', error.response?.data || error.message);
    throw error;
  }
}

// Usage
translateText('Hello, how are you?')
  .then(translation => console.log(translation));

Error Handling Example

async function translateWithRetry(text, maxRetries = 3) {
  for (let i = 0; i < maxRetries; i++) {
    try {
      const response = await fetch('https://french-translation.onrender.com/translate', {
        method: 'POST',
        headers: { 'Content-Type': 'application/json' },
        body: JSON.stringify({ text })
      });
      
      if (!response.ok) {
        throw new Error(`HTTP ${response.status}`);
      }
      
      const data = await response.json();
      return data.translated_text;
      
    } catch (error) {
      if (i === maxRetries - 1) throw error;
      await new Promise(r => setTimeout(r, 1000 * Math.pow(2, i)));
    }
  }
}
